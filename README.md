Creative Lapcare is a responsive e-commerce web application developed for showcasing and managing laptop spare parts and accessories. The system allows users to browse various products such as laptop displays, keyboards, batteries, and other components with a user-friendly interface.

## 🚀 Features
- Responsive and user-friendly web design
- Product listing and category navigation
- Detailed product view with images and specifications
- Organized menu for laptop spares and accessories
- Clean UI with easy navigation

## 🛠️ Technologies Used
- Frontend: HTML, CSS, Bootstrap, JavaScript
- Backend: Python
- Database: PHP, SQL
- Tools: Figma (for UI/UX design)

- ## 📂 Project Structure
- index.html – Home page
- display.html – Product category page
- product.html – Product listing page
- product-details pages – Detailed product view
